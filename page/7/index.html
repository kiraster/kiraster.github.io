我的一些blog常用小操作

自己日常操作的一些小命令和插件。

bat启动 hexo 本地服务

前提条件：已完成博客搭建。

@echo off
start D:\Program" "Files\Git\git-bash.exe --cd=D:/blog/blog -c "hexo clean && hexo g && hexo s"
  1. 新建后缀为bat的脚本文件
  2. 复制以上命令粘贴（注意修改路径 ），保存
  3. 双击此脚本运行的结果相当于在你的 blog 路径下右键 “Git Bash Here” 并输入 “hexo clean && hexo g && hexo s” 运行.
